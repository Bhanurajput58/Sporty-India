# Sporty India

**Sporty India** is a community-driven platform designed to empower athletes across India. It connects athletes with recruiters, coaches, educators, and professionals, creating a vibrant ecosystem for nurturing sporting talent.

# Features

- **User-Friendly Navigation**: Intuitive header with dropdown support for mobile devices.
- **Dynamic Statistics**: Live data on registered athletes, legitimacy ratings, and job opportunities.
- **Custom Accordion**: FAQ section to resolve common user queries.
- **Powerful Search**: Quickly find athletes, sports, or ratings with built-in search functionality.
- **Responsive Design**: Seamless experience across all devices and screen sizes.

---

# Technologies Used

- **React** – For building interactive UIs.
- **Next.js** – Framework for server-side rendering and static generation.
- **Framer Motion** – For fluid animations and transitions.
- **React Icons** – Icon library for seamless integration of popular icons.
- **Tailwind CSS** – Utility-first CSS framework for fast and responsive styling.

# Getting Started

To run **Sporty India** locally on your machine:

# 1. Clone the Repository

```bash
git clone https://github.com/Bhanurajput58/Sporty-India.git
cd Sporty-India
````

### 2. Install Dependencies

```bash
npm install
```

### 3. Run the Development Server

```bash
npm run dev
```

### 4. View in Browser

Open your browser and go to:
[http://localhost:3000](http://localhost:3000)

## Usage

Once the server is running:

* Explore athletes listed on the platform.
* Use the search bar to filter based on sport, name, or rating.
* Navigate through FAQs to understand more about the platform.
* Try it on mobile and desktop to experience the responsive design.

## Contributing

We welcome contributions! Here's how you can get started:

### How to Contribute

1. **Fork the Repository**
   Click the "Fork" button on GitHub to create your own copy.

2. **Clone Your Fork**

   ```bash
   git clone https://github.com/YOUR_USERNAME/Sporty-India.git
   cd Sporty-India
   ```

3. **Create a Branch**

   ```bash
   git checkout -b feature/your-feature-name
   ```

4. **Make Changes**
   Implement your feature or fix bugs while maintaining code quality.

5. **Commit Changes**

   ```bash
   git commit -m "Add feature: your-feature-name"
   ```

6. **Push to GitHub**

   ```bash
   git push origin feature/your-feature-name
   ```

7. **Submit a Pull Request**
   Go to the original repository and click **"New Pull Request"**. Describe your changes clearly.

### Guidelines

* Keep code clean and consistent with the existing style.
* Use meaningful commit messages.
* Open an issue to discuss ideas before major changes.
